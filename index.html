What is React JS

  • React is a JavaScript library
    that is used to build user interface.

  • React was originally intended for the web, for building websites, but later the React Native platform appeared, which was already intended for mobile devices.

  • React presents the perfect tool for building scalable web applications.

Virtual DOM

  • The entire structure of a web page can be represented using the DOM (Document Object Model) - an organization of html elements that we can manipulate, modify, delete or add new ones.

  • JavaScript is used to interact with the DOM.

  • The virtual DOM represents a lightweight copy of the regular DOM. And a distinctive feature of React is that this library works with the virtual DOM, and not the usual one.

Virtual DOM

  • If an application needs to know information about the state of elements, then the virtual DOM is accessed.

  • If elements of a web page need to be changed, the changes are first made to the virtual DOM. Then the new state of the virtual DOM is compared with the current state. And if these states are different, then React finds the minimum amount of manipulation that is necessary before updating the real DOM to a new state and performs them.

Other features of React

  • we can create individual components and then easily transfer them from project to project.

  • we use JSX

  • JSX represents a combination of JavaScript and XML code and provides a simple and intuitive way to define the visual interface code.

  • also to compile the code, we need the Babel compiler, which allows us to work with the code according to the standard ES6 (ES2015).

  • Project template engines are available.

Other features of React

  • The concept of View from MVC is used.

  • React allows you to create interfaces. In the well-known pattern Model-View-Controller React is closest to the user. It is responsible for presenting data, receiving and processing user input. Where and in what form you store data, how you communicate with the storage, is your business. React is just a View of your application.

Other features of React

  • React is built on the reactive programming paradigm. This declarative approach suggests describing data as a set of statements or formulas. Changing one of the parameters leads to an automatic recalculation of all dependencies

Installing from NPM

  • The React manual also recommends using the react and react-dom npm packages. To install them on your system, run the following command on the bash command line in your project directory, or first create a new directory and cd into it.

  • $ npm install --save react react-dom $ browserify -t babelify main.js -o bundle.js You should now be able to see the React installation inside the node_modules directory.

What is JSX?

  • JSX is an XML / HTML syntax that is used to render HTML from JavaScript code. React converts JSX to native JavaScript for the browser and with the provided tools you can convert HTML code of existing sites to JSX

What is JSX?

  • JSX makes it easier to write code because it is very similar to writing familiar HTML, but only from within JavaScript. When combined with Node, this provides a very consistent workflow.
